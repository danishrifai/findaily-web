# FinDaily Web

Aplikasi web untuk mengatur keuangan dan jadwal harian.

Dibuat menggunakan HTML, CSS, dan JavaScript.

Fitur saat ini:
- Melihat saldo, pemasukan, pengeluaran
- Melihat pengeluaran utama per kategori
- Melihat jadwal tugas hari ini
